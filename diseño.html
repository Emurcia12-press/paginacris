Diseño Web

¿Qué es una Página Web?

Una página web es un documento digital accesible desde un navegador web que contiene texto, imágenes, videos, enlaces y otros elementos. Está diseñada para ofrecer contenido informativo, comercial, educativo, entre otros.

Breve Historia de las Páginas Web

Las páginas web nacieron en 1991, cuando Tim Berners-Lee creó el primer sitio en CERN. Desde entonces, han evolucionado desde documentos simples en HTML hasta complejas plataformas interactivas con HTML5, CSS3, y JavaScript.

Tipos de Páginas Web

Tipo Descripción Ejemplo
Estáticas Contenido fijo, no cambia sin intervención manual. Portafolios personales.
Dinámicas Contenido generado automáticamente desde una base de datos. Redes sociales, blogs.
Responsivas Se adaptan a cualquier tamaño de pantalla. Sitios modernos y móviles.

Estructura Básica de una Página Web

Una página web se construye principalmente con tres tecnologías:

  • HTML (HyperText Markup Language): estructura del contenido.
  • CSS (Cascading Style Sheets): diseño visual (colores, fuentes, espaciado).
  • JavaScript: interactividad y funcionalidades dinámicas.

HTML (-Hyper Text Markup Language-)

HTML es un lenguaje utilizado para la creación de páginas web, está compuesto por etiquetas que el navegador interpreta para mostrar imágenes, texto, listas, tablas, etc. HTML dispone de etiquetas que permiten controlar imágenes, hipervínculos, saltos de línea, tablas, etc.

HTML5

Es el lenguaje de marcado con el que se crea la estructura y el contenido de las páginas web, a su vez, es la nueva versión de HTML. como dato curioso, la versión definitiva de la quinta revisión del estándar se publicó en octubre de 2014.

Tablas en Documentos HTML

Las tabla se utilizan para agrupar información en columnas y filas. En versiones de HTML, las tablas se utilizaban para la maquetación de los sitios, con la aparición de los CSS esta técnica ya no se utiliza.

CSS3

Es un lenguaje de hojas de estilos que ayuda a controlar el aspecto o presentación de los documentos electrónicos creados con HTML y XHTML. Al utilizar CSS se separan los contenidos de la presentación del documento. CSS (Cascading Style Sheets) trabaja junto con HTML para proveer estilos visuales a los elementos del documento como: tamaño, color, fondo, bordes, etc ....

CSS es un complemento desarrollado para superar las limitaciones y reducir la complejidad de HTML.

CSS tiene un set predeterminado de propiedades que permiten sobrescribir los estilos provistos por navegadores y obtener la organización deseada. Estas propiedades no son específicas, tienen que ser combinadas para formar reglas que después serán usadas para agrupar cajas y obtener la correcta disposición en pantalla.

La combinación de estas reglas es normalmente llamada Modelo o Sistema de Disposición. Todas estas reglas aplicadas juntas constituyen lo que se llama un Modelo de Caja.

Al crear una página web, se utiliza en primer lugar el lenguaje HTML/XHTML para marcar los contenidos, es decir, para designar la función de cada elemento dentro de la página: párrafo, titular, texto destacado, tabla, lista de elementos, etc.

Aplicar Reglas CSS3 utilizando Dreamweaver

Como regla general segun W3C (Word Wide Web Consortium) exige que el archivo HTML solamente muestre datos organizados y estructurados con etiquetas de marcación HTML, mientras que la parte de formato debe realizarse mediante los estilos CSS única y exclusivamente.

Para aplicar reglas en cualquier aplicación, bastará con crear un archivo con la extensión .css y escribir sobre el las mismas.

Para crear una archivo se estilos CSS en Dreamweaver, realizar las siguientes instrucciones:

  1. Crear el diseño HTML.
  2. Crear un archivo de estilos CSS y vincularlo con el documento.
    • Abrir el Panel Disenador de CSS dando click en el Menu Ventana y a continuación sobre Diseñador de CSS o presionando la combinación de teclas Mayús (shift) + F11.
    • En la sección Fuentes del panel Diseñador de CSS dar click en el botón Agregar origen CSS (+) y a continuación dar click sobre Crear nuevo archivo CSS.
    • Escribir el nombre del archivo con la extension .css y dar click en el Boton Aceptar.
  3. Crear las reglas
    • En el código dar click sobre la etiqueta que encierra los elementos a los cuales les desea aplicar un estilo. En nuestro ejemplo, dar click sobre body y en el area Selectores del panel Diseñador de CSS dar click sobre el Boton Añadir Selector.
    • Seleccionar el selector y añadir las propiedades.
    • Realizar las mismas actividades para crear las reglas por cada objeto del documento. El código del archivo seria .css.

Apache

Es un servidor web HTTP de código abierto para la creación de páginas y servicios web, a su vez, es un servidor multiplataforma, gratuito y robusto.

WampServer

Es un entorno de desarrollo web para Windows con el que se pueden crear aplicaciones web con Apache, PHP y bases de datos MySQL database. Tambien incluye PHPMyAdmin para manejar las bases de datos de MySQL.

PHPMyAdmin

Es un programa gratuito escrito en PHP que permite manejar la administración de MySQL a través de páginas web, utilizando Internet. Está disponible bajo la licencia GPL (General Public License) y en más de 50 idiomas.

Diseño Web Responsivo

El diseño responsivo permite que un sitio se vea bien en computadoras, tablets y teléfonos. Esto se logra usando media queries en CSS.

Accesibilidad Web

Es el diseño de sitios web que pueden ser utilizados por personas con discapacidades visuales, auditivas o motoras. Incluir texto alternativo en imágenes, buena estructura de encabezados y navegación clara son prácticas esenciales.

Buenas Prácticas en Diseño Web

  • Diseño limpio y minimalista.
  • Colores contrastantes para una buena lectura.
  • Optimización para móviles.
  • Uso de etiquetas semánticas en HTML.
  • Imágenes optimizadas para no ralentizar el sitio.

Conclusión

El diseño de páginas web es una combinación entre arte y tecnología. Nos permite comunicar, educar, vender y entretener a través de la pantalla. Aprender esta habilidad abre puertas al mundo digital, al desarrollo profesional y a la creatividad sin límites.
